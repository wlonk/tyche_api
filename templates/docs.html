Documentation

Ask Tyche for help in Discord via ?help (or whatever prefix is configured for your server).

Tyche can control music, roll dice, and grant or revoke roles. Music commands include ?play, ?pause, ?resume, ?stop, and ?vol. Dice commands include ?roll. Role commands include ?role and ?unrole.
